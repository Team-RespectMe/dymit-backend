# AGENTS.MD

## Project Overview
이 프로젝트는 주어진 웹사이트 페이지를 파싱하여 DYMIT에서 제공할 스터디 그룹 홍보 페이지에 외부 스터디 정보를 제공하는 것이 목적입니다.  
기본적으로 cli 상태로 동작을 상정합니다. 
파이썬으로 작성해야 합니다. 
파이썬 패키지 관리자는 uv를 이용합니다.  
파이썬 3.12 버전을 기준으로 작성되었습니다.  

## Setup Command
- Install Dependencies
```bash
uv sync
```
- Run Application
```bash
uv run python main.py
```
- Run Tests (pytest)
```bash
uv run pytest
```

## Code Style
- Python PEP8 준수
- 매개변수와 반환형에 항상 타입 힌트를 다십시오. 

## Rules
- 한 파일의 최대 크기는 200줄로 제한합니다. 그 이상은 파일을 분리하세요.
- 비지니스 로직 작성 시 테스트 코드를 함께 수정하는 것은 금지됩니다.
- 비지니스 로직 작성 후 반드시 테스트를 수행해하십시오, 실패 시 테스트 코드 수정을 금지합니다.
- 프로젝트 구조를 앱과 테스트 디렉터리를 분리하여 운영합니다.
- 스터디 모집 정보를 모을 사이트는 여러가지이므로 파싱은 전략 패턴을 사용합니다. 별도의 패키지로 분리하세요.
- 사이트 별 파싱 순서는 HTML 태그 + id 를 맵 형태로 주어진 깊이까지 들어가 파싱을 수행하면 리스트 형태로 목록을 가져와야합니다.
- 각 엘리먼트에서 가져올 데이터와 이를 앱 내부에서 다룰 공통 인터페이스로 매핑합니다. 해당 정보는 parsing.yaml 에 정의하고 세션 중 추가되는 사항을 바로 해당 파일에 업데이트하세요
- main.py 에서는 기본적으로 parsing.yaml 을 반드시 로드합니다.
- 각 파일 상단에는 어떤 기능을 하는 파일인지 항상 기능을 작성하며, 매개변수와 반환형에 대한 정보를 명시합니다.
